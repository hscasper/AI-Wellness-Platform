    ```yaml
    healthcheck:
      test: ['CMD-SHELL', 'curl -f http://localhost:8080/api/auth/health || exit 1']
      interval: 10s
      timeout: 5s
      retries: 5
      start_period: 15s
    ```

    Insert this block after line 254 (after `ASPNETCORE_ENVIRONMENT: 'Production'`), before the VOLUMES section comment.
  </action>
